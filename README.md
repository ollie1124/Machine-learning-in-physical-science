# Machine-learning-in-physical-science
This repository contains the machine learning applications for physical science, outlined by Professor Tuan Do at UCLA and completed by myself.  

# Gaussian_processes: Time Series Forecasting (CO2 Prediction)
Built Gaussian Process regression models (scikit-learn) with composite kernels to model atmospheric CO2 trends
(Mauna Loa dataset). Generated long-term forecasts with uncertainty estimates validated against historical data.

# Galaxy clustering and dwarf galaxy remnant detection: Unsupervised Learning for Astronomical Data
Applied K-means and Gaussian Mixture Models (scikit-learn) to Gaia survey data to identify stellar substructures. Detected candidate dwarf galaxy remnants using velocity and metallicity features.

# Deep Learning for COVID-19 Forecasting 
Designed and trained neural networks (Keras) to model global COVID-19 case trends across regions.

# Particle Physics Classification using random forest
Engineered features from particle collision data and trained a Random Forest classifier to identify top quark events. Achieved 96% F1-score and 97% precision on test data.
